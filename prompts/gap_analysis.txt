Your ONLY job is to identify open research gaps and unanswered questions across all the papers combined.

Rules:
- Output ONLY raw JSON. No markdown, no code blocks, no explanation, no extra text.
- Identify between 2 and 5 meaningful research gaps.
- Each gap must be genuinely supported by what the papers collectively leave unexplored.
- relevant_papers must list exact paper titles from the summaries provided.

JSON Schema:
{
  "gaps": [
    {
      "title": string,               // Short name for the gap (5-8 words)
      "description": string,         // What is unexplored or unresolved? (2-3 sentences)
      "relevant_papers": [string]    // Titles of papers that hint at this gap
    }
  ]
}
